Acronym Enlightenment
Crowdsourced & Easy

Did Jim in Finance just send you a direct message using another confusing accounting acronym?
Don't ask Jim. Ask Nym!

Use a slash command ("/nym EBITDA") and let Nym shed light on the acronym in a private message that only you'll see.


Did Tim in Customer Service ask about an acronym used by the boss in a public channel?
Wanna help Tim? Ask Nym!

Mention Nym with one or more acronyms ("@nym WTH is GTM?") to let Nym define the known acronyms in a threaded reply that everyone can see.


Did Kim in Engineering ask you to define a new acronym you used in a digital standup?
Don't teach Kim. Teach Nym!

When undefined acronyms are used, Nym will automatically ask the user to define it. The more users define acronyms, the smarter Nym gets!
